# Smart Traffic Light Controller

## Description
This project demonstrates a Smart Traffic Light Controller using Arduino in Tinkercad. The system controls Red, Yellow, and Green LEDs to simulate a traffic signal. A push button is used to change the traffic light sequence manually.

## Components Used
- Arduino Uno
- Red LED
- Yellow LED
- Green LED
- Push Button
- 220Ω Resistors
- Breadboard
- Jumper Wires
- Tinkercad Circuits

## Working
1. The Green LED is ON by default.
2. When the push button is pressed, the Green LED turns OFF.
3. The Yellow LED turns ON for a few seconds.
4. The Red LED turns ON to stop traffic.
5. After the delay, the Green LED turns ON again, completing one traffic signal cycle.
